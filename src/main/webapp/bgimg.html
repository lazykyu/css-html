background 배경

  1. background-color:색상; 배경색
  2. background-image:url("이미지경로"); 배경이미지

background-image의 속성들

  • background-repeat: ; 배경이미지 반복 지정
    • repeat 기본값. 양사방 반복
    • repeat-x 가로로만 반복
    • repeat-y 세로로만 반복
    • no-repeat 반복 안함
  • background-size: ; 배경이미지 크기 지정
    • cover 이미지 하나로만 영역을 덮음.
      (이미지가 잘려나갈 수도)
    • contain 이미지를 가로나 세로에 맞춰 최대한 영역을 덮음. (이미지 하나가 온전히 나옴.
      반복되거나 빈 공간이 생길 수도)
    • Xn Yn 가로 세로 순으로 직접 크기 지정.
  • background-position: ; 배경이미지 위치 지정
    • 방향값 (left/center/right) (top/center/bottom) 중에서 수평 수직 순으로 조합하여 지정.
    • Xn Yn 가로 세로 순으로 직접 위치값 지정.
  • background-attachment: ; 배경이미지 스크롤여부 지정
    • scroll 기본값. 자기 영역 스크롤바에는 고정, 브라우저 스크롤바에는 스크롤
    • fixed 자기 영역 스크롤바와 브라우저 스크롤바에 모두 고정
    • local 자기 영역 스크롤바와 브라우저 스크롤바에 모두 스크롤

배경이미지 2개 적용

선택자 {background-image: url("A"), url("B");} 기타 다른 배경 속성도 ,로 값을 추가함

특정 영역에서의 background-attachment

  1. background-attachment:scroll;
    기본값. 자기 영역 스크롤바에는 고정, 브라우저 스크롤바에는 스크롤
  2. background-attachment:fixed;
    자기 영역 스크롤바와 브라우저 스크롤바에 모두 고정
  3. background-attachment:local;
    자기 영역 스크롤바와 브라우저 스크롤바에 모두 스크롤
