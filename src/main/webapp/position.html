position - 요소의 위치 지정 방식

  • position:static; 기본값. 위치 변동 불가
  • position:relative; 소스 순서대로의 위치에서 ~변경
  • position:absolute; 상위요소를 기준으로 ~변경
  • position:fixed; 브라우저를 기준으로 ~에 고정
  • position:sticky; relative와 fixed의 믹스

position:sticky; relative와 fixed의 믹스

position:sticky는 상위에 다른 요소가 있으면 적용 불가.
* color값의 rgba(n,n,n,n)는 컬러에 투명도를 지정. a의 값을 0~1사이로 적용 (0일 때 완전투명)

opacity:n; 요소의 투명도 조절

요소 자체의 불투명도를 0~1사이의 값으로 조정.
0일 때 완전투명, 1일 때 불투명.
