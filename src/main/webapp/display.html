display 요소가 나타나는 방식(성격) 지정

tag를 box-model이라 하며 크게 inline과 block으로 나뉨.
inline은 한 줄에 나란히 오며(자동으로 줄이 안바뀜) 크기 적용 불가, 상하 margin 적용 불가.
block은 한 줄에 나란히 올 수 없고(자동으로 줄이 바뀜) 크기와 양사방 margin 모두 가능.

display

  • display:inline; block인 요소를 inline로 바꿈
  • display:block; inline인 요소를 block로 바꿈
  • display:inline-block; 어떤 요소든지 inline과 block의 성격을 합친 요소로 바꿈.
    한 줄에 나란히 오면서 크기와 양사방 margin 모두 가능
  • display:none; 요소를 화면에서 숨김

display:inline; block인 요소를 inline로

block인 p

block인 div
block인 form

display:block; inline인 요소를 block로

inline인 a inline인 b inline인 span

display:inline-block; 어떤 요소든지 inline과 block의 성격을 합친 요소로

block인 p

block인 div
block인 form
inline인 a inline인 b inline인 span

display: none; 요소를 화면에서 숨김(영역까지 같이 사라짐)

block 인 p

block 인 div
block 인 form
inline 인 a inline 인 b inline 인 span

visibility; (영역을 남기고) 요소를 화면에서 숨김

block 인 p

bolck 인 div
block 인 form
inline 인 a inline 인 b inline 인 span
