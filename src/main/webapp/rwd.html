rwd(Responsive Web Design)

mobile first가 원칙~! 작은 화면을 기준으로 기본 작업 후 각 사이즈별로 미디어쿼리로 소스를 추가함

Viewport

뷰포트는 웹페이지에서 사용자가 볼 수 있는 영역입니다.
태블릿과 휴대폰에서 브라우저는 전체 웹 페이지를 화면에 맞게 축소되므로 문제가 됨.
=> 뷰포트 설정
<meta name="viewport" content="width=device-width, initial-scale=1.0">
(장치의 화면 너비를 따르도록 페이지 너비를 설정 브라우저에서 페이지를 처음 로드할 때 초기 확대/축소 수준을 설정)

Media Queries 구문

	@media 장치종류 and (조건){
     css선택자 {속성:속성값;}
	}

Media Queries의 장치종류

  • all - 모든 장치
  • print - 프린터
  • screen - screen으로 나타나는 computer들 (tablets, smart-phones etc.)
  • speech - 음성낭독기

Media Queries의 조건들

  • width and height of the viewport - 뷰포트의 너비나 높이
  • width and height of the device - 장치의 너비나 높이
  • orientation (is the tablet/phone in landscape or portrait mode?) - 화면 방향
  • resolution - 해상도
