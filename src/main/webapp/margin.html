margin- 요소의 외부 여백

margin의 방향별 지정 속성

  • margin-top:n; 위에만 적용
  • margin-right:n; 우측에만 적용
  • margin-bottom:n; 아래만 적용
  • margin-left:n; 좌측에만 적용

margin의 값 갯수별 적용법

  • margin:n; 양사방 값 통일
  • margin:n n; 상하, 좌우 순으로 적용
  • margin:n n n; 상, 좌우, 하 순으로 적용
  • margin:n n n n; 상, 우, 하, 좌 순으로 적용
[center] margin-left:auto; margin-right:auto; 또는 margin:n auto;

inline요소는 상하 margin이 적용 안됨, 좌우는 인접 요소 각각 적용.
